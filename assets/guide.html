1、 Install client software in smartphone

Please download relevant client according to the following two methods, and install it after successfully download.

Method one: Use your smartphone to scan the relevant QR code on the package box in order to download client.

Method two: iPhone users can search client software with keyword ‘WiWo’ in App Store and download it; Android users can search client software with keyword ‘WiWo’ in Google Play or other application markets and download it.

2、 Connect the socket with WiFi router

If you use the socket for the first time, please connect the socket with WiFi router. Firstly, please ensure your smartphone has been connected with WiFi router, and hold on the on-off key until power on, if the backlight of the on-off key flashes quickly, which means the socket is in config mode; if not, then please power off the socket and power on again according to the method mentioned before. Secondly, open the client in your smartphone, click 【Smart Setup】 in the 【Settings】 interface, then input the WiFi password and click 【Setup new socket】. The backlight of the on-off key flashes slowly means the socket is connecting with the WiFi router, the backlight of the on-off key keeps on means the socket connected with the WiFi router.

3、 Add devices

If the socket has connected with the WiFi router, please click 【Smart Setup】 in the 【Settings】 interface to enter next operation interface, you can add devices by click the 【Search socket】 button which under the 【Setup new socket】 button, if success, you can find the socket added to the 【Devices】 interface.

4、 Modify devices info

Please enter the 【Customise】 interface first, click the device you want to modify to enter the next operation interface, and then you can modify the device name(no more than 16 characters) or icon, it's convenient for you to find the device you want to control. In addition, you can lock/unlock device in the interface, and then nobody can find the device after locked. Only the user who have added the device before locked can control the device, other people only can add the device in the local network after unlocked.

5、 Modify remote password

Please enter the 【Customise】 interface first, click the device you want to modify to enter the next operation interface, and then click the 【Modify Remote Password】 button. Please input the old and new password(The default remote password is 888888, new password can't exceed 12 characters) in the MessageBox, if modified successfully, the users whose remote password are different from that you set can't control the socket.

6、 Delete devices

Please enter the 【Customise】 interface first, click the device you want to delete to enter the next operation interface, and then click the 【Delete】 button at the top-right corner to delete the device.

7、 Set timing

Please enter the 【Timing】 interface, click 【Timing】, and then click the 【Add】 button to add a timing. Click 【Repeat】 to choose repeat mode, such as single, every Monday, every Sunday and so on. Click 【Time】 to set time according to your need. Click 【Devices】 to choose the device you want to control and set the on/off action you want the device to do. Click the 【Save】 button to save the timing info you set, if saved successfully, the timing info will display on the 【Timing】 interface.

8、 Set countdown

Please enter the 【Timing】 interface, click 【Countdown】 and then choose the device you want to set, you can set time by radio button or time slider bar. If you have set countdown task, you can click the 【Cancel】 button to cancel the task, and can get and modify the countdown info in this interface.

9、 Restore factory settings

Hold on the on-off key before power on, loosen the key at least 3 seconds later after power on, if the backlight of the on-off key flashes slowly, means restore successfully, if not, please try again.
